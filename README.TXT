# ToDoList
Trabalho final da disciplina de Linguagem de programação II.
Lista de atividades do mes implementada em Java,ultilizando o swing para interface grafica.

## Funcionalidades
* 4 tipos de atividades (Compromisso familiar, compromisso profissional, evento e lazer)
* Adiciona atividades
* Remove atividades
* Altera a data de uma atividade

## Limitações
* Mostra somento as atividades do mes.

### Execução
* Executar o arquivo "toDoList/terceira_avaliacao_lp2/src/lp2/visao/TelaPrincipal.java"
